# AGENTS.md

## Язык
- ВСЁ общение и артефакты — на русском:
  - сообщения коммитов
  - названия и описания PR
  - описания задач/issue (если создаёшь)
  - комментарии в коде (//, /* */), TODO/FIXME
  - комментарии к ревью / ответы в обсуждениях
- Исключения: код (имена переменных/функций/классов), ключевые слова языков, имена файлов, команды, названия библиотек — как в оригинале (обычно английский).

## Коммиты
- Формат: <type>(scope): <сообщение по-русски>
- type: feat | fix | refactor | perf | test | docs | chore | ci | build
- Примеры:
  - feat(api): добавил фильтр published для публичных модулей
  - fix(auth): исправил проверку JWT в админке
  - refactor(db): вынес индексы в отдельный файл

## Pull Request
- Заголовок PR — по-русски, коротко и по делу.
- Описание PR — по шаблону:
  - Что сделано
  - Почему
  - Как проверить
  - Риски/миграции
  - Скриншоты (если UI)

## Комментарии в коде
- Пишем по-русски, конкретно: что/почему, а не “очевидно”.
- TODO всегда с контекстом: TODO(когда/почему/ссылка на задачу).
